Class Faraday::Adapter::Rack
In: lib/faraday/adapter/rack.rb
Parent: Faraday::Adapter

Sends requests to a Rack app.

Examples 

  class MyRackApp
    def call(env)
      [200, {'Content-Type' => 'text/html'}, ["hello world"]]
    end
  end

  Faraday.new do |conn|
    conn.adapter :rack, MyRackApp.new
  end
